# Black Excellence Archives

### Black Vision. Black Brilliance. Black Legacy.

Black Excellence Archives is a cinematic digital archive dedicated to preserving and celebrating the impact of Black innovators, visionaries, artists, scientists, writers, engineers, and historical figures across mathematics, science, literature, history, the arts, and technology. Inspired by a Netflix-style browsing experience combined with the educational depth of digital archives and museum storytelling, the platform was designed to create a more immersive and engaging way to explore Black excellence throughout history through curated biographies, categorized collections, dynamic profile pages, and a premium black-and-gold user experience.

---

## 🎬 Key Features

* Netflix-inspired cinematic browsing experience
* Dynamic category-based archive system
* Search and filter functionality
* Responsive black-and-gold user interface
* Related Visionaries recommendation sections
* Smooth animations and immersive transitions
* Curated educational storytelling experience
* Archive-style navigation and discovery

---

## 💼 Tech Stack

* **Next.js** – Powers the application structure, routing system, page rendering, and overall full-stack architecture for the archive experience.
* **React** – Builds reusable UI components including profile cards, category rows, navigation elements, and dynamic content sections.
* **TypeScript** – Adds static typing and improved code reliability to create a more scalable, maintainable, and organized codebase.
* **Tailwind CSS** – Handles the cinematic black-and-gold styling system, responsive layouts, spacing, gradients, and visual consistency across the platform.
* **Framer Motion** – Creates smooth animations, hover effects, page transitions, and immersive motion interactions throughout the archive.
* **Dynamic Routing** – Generates individual profile pages dynamically for all innovators while supporting scalable category-based navigation and archive organization.
